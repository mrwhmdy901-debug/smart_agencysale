3. How to Edit

  • Open the project in Visual Studio Code.
  • Open index.html or index-rtl.html to edit content.
  • Change texts, images, and icons directly from HTML.
  • All colors and styles are inside style.css.

4. Live Preview

Use the Live Server extension in VS Code to preview the website locally.
